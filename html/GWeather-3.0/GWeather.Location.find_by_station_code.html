GWeather.Location.prototype.find_by_station_code

function find_by_station_code(station_code: String): GWeather.Location {
    // Gjs wrapper for gweather_location_find_by_station_code()
}

Retrieves the weather station identifier by station_code. Note that multiple instances of the same weather station can exist in the database, and this function will return any of them, so this not usually what you want.

See GWeather.Location.prototype.deserialize to recover a stored GWeather.Location.

station_code

a 4 letter METAR code

Returns

a weather station level GWeather.Location for station_code, or null if none exists in the database.
