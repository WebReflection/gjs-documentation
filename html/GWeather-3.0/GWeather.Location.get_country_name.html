GWeather.Location.prototype.get_country_name

function get_country_name(): String {
    // Gjs wrapper for gweather_location_get_country_name()
}

Gets the country name of loc. For a GWeather.LocationLevel.country location, this is equivalent to GWeather.Location.prototype.get_name. For a GWeather.LocationLevel.region and GWEATHER_LOCATION_WORLD location it will return null. For other locations it will find the parent GWeather.LocationLevel.country and return its name.

Returns

loc's country name, or null
