GWeather.Location.prototype.deserialize

function deserialize(serialized: GLib.Variant): GWeather.Location {
    // Gjs wrapper for gweather_location_deserialize()

This call undoes the effect of GWeather.Location.prototype.serialize, that is, it turns a GLib.Variant into a GWeather.Location. The conversion happens in the context of world (i.e, for a city or weather station, the resulting location will be attached to a administrative division, country and region as appropriate).

serialized

the GLib.Variant representing the GWeather.Location

Returns

the deserialized location.
