Gio.ThemedIcon.use_default_fallbacks

ThemedIcon.use_default_fallbacks: Boolean (Read / Write / Construct Only)

Whether to use the default fallbacks found by shortening the icon name at '-' characters. If the "names" array has more than one element, ignores any past the first.

For example, if the icon name was "gnome-dev-cdrom-audio", the array would become |[<!-- language="C" --> { "gnome-dev-cdrom-audio", "gnome-dev-cdrom", "gnome-dev", "gnome", NULL }; ]|
