Gio.Action.prototype.get_state

function get_state(): GLib.Variant {
    // Gjs wrapper for g_action_get_state()
}

Queries the current state of action.

If the action is not stateful then null will be returned. If the action is stateful then the type of the return value is the type given by Gio.Action.prototype.get_state_type.

The return value (if non-null) should be freed with GLib.Variant.prototype.unref when it is no longer required.

Since 2.28

Returns

the current state of the action
