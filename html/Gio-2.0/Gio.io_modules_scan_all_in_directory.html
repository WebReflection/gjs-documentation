Gio.io_modules_scan_all_in_directory

function io_modules_scan_all_in_directory(dirname: String): void {
    // Gjs wrapper for g_io_modules_scan_all_in_directory()
}

Scans all the modules in the specified directory, ensuring that any extension point implemented by a module is registered.

This may not actually load and initialize all the types in each module, some modules may be lazily loaded and initialized when an extension point it implementes is used with e.g. Gio.IOExtensionPoint.prototype.get_extensions or Gio.IOExtensionPoint.prototype.get_extension_by_name.

If you need to guarantee that all types are loaded in all the modules, use Gio.io_modules_load_all_in_directory.

Since 2.24

dirname

pathname for a directory containing modules to scan.
