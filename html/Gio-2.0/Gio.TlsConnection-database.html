Gio.TlsConnection.database

TlsConnection.database: Gio.TlsDatabase (Read / Write)

The certificate database to use when verifying this TLS connection. If no cerificate database is set, then the default database will be used. See Gio.TlsBackend.prototype.get_default_database.
