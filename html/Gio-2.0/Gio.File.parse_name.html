Gio.File.parse_name

Constructs a Gio.File with the given parse_name (i.e. something given by Gio.File.prototype.get_parse_name). This operation never fails, but the returned object might not support any I/O operation if the parse_name cannot be parsed.

parse_name

a file name or path to be parsed
