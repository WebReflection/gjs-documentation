Gio.ActionGroup.prototype.change_action_state

function change_action_state(action_name: String, value: GLib.Variant): void {
    // Gjs wrapper for g_action_group_change_action_state()
}

Request for the state of the named action within action_group to be changed to value.

The action must be stateful and value must be of the correct type. See Gio.ActionGroup.prototype.get_action_state_type.

This call merely requests a change. The action may refuse to change its state or may change its state to something other than value. See Gio.ActionGroup.prototype.get_action_state_hint.

If the value GVariant is floating, it is consumed.

Since 2.28

action_name

the name of the action to request the change on

value

the new state
