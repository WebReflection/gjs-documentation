Gio.Proxy

const Gio = imports.gi.Gio;

let proxy = new Gio.Proxy();

A Gio.Proxy handles connecting to a remote host via a given type of proxy server. It is implemented by the 'gio-proxy' extension point. The extensions are named after their proxy protocol name. As an example, a SOCKS5 proxy implementation can be retrieved with the name 'socks5' using the function Gio.IOExtensionPoint.prototype.get_extension_by_name.
