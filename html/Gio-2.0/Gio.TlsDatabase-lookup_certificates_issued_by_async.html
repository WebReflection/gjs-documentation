Gio.TlsDatabase::lookup_certificates_issued_by_async

function vfunc_lookup_certificates_issued_by_async(issuer_raw_dn: ByteArray, interaction: Gio.TlsInteraction, flags: Gio.TlsDatabaseLookupFlags, cancellable: Gio.Cancellable, callback: Gio.AsyncReadyCallback): void {
}

Asynchronously lookup certificates issued by this issuer in the database. See Gio.TlsDatabase.prototype.lookup_certificates_issued_by for more information.

The database may choose to hold a reference to the issuer byte array for the duration of of this asynchronous operation. The byte array should not be modified during this time.

Since 2.30

issuer_raw_dn

a GLib.ByteArray which holds the DER encoded issuer DN.

interaction

used to interact with the user if necessary

flags

Flags which affect the lookup operation.

cancellable

a Gio.Cancellable, or null

callback

callback to call when the operation completes
