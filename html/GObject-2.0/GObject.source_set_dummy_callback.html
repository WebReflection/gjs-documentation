GObject.source_set_dummy_callback

function source_set_dummy_callback(source: GLib.Source): void {
    // Gjs wrapper for g_source_set_dummy_callback()
}

Sets a dummy callback for source. The callback will do nothing, and if the source expects a #gboolean return value, it will return true. (If the source expects any other type of return value, it will return a 0/null value; whatever GObject.Value.init initializes a GObject.Value to for that type.)

If the source is not one of the standard GLib types, the @closure_callback and @closure_marshal fields of the GLib.SourceFuncs structure must have been filled in with pointers to appropriate functions.
