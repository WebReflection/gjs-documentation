GObject.ParamSpecTypeInfo.n_preallocs

ParamSpecTypeInfo.n_preallocs: Number(guint16) (Read / Write)

Prior to GLib 2.10, it specified the number of pre-allocated (cached) instances to reserve memory for (0 indicates no caching). Since GLib 2.10, it is ignored, since instances are allocated with the <link linkend="glib-Memory-Slices">slice allocator</link> now.
