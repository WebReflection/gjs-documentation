GObject.Binding.prototype.unbind

function unbind(): void {
    // Gjs wrapper for g_binding_unbind()
}

Explicitly releases the binding between the source and the target property expressed by binding.

This function will release the reference that is being held on the binding instance; if you want to hold on to the GObject.Binding instance after calling GObject.Binding.prototype.unbind, you will need to hold a reference to it.

Since 2.38
