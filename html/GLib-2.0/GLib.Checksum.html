GLib.Checksum

const GLib = imports.gi.GLib;

let checksum = new GLib.Checksum(checksum_type: GLib.ChecksumType);

An opaque structure representing a checksumming operation. To create a new GChecksum, use GLib.Checksum.new. To free a GChecksum, use GLib.Checksum.prototype.free.
