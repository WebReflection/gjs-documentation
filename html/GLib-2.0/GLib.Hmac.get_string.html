GLib.Hmac.prototype.get_string

function get_string(): String {
    // Gjs wrapper for g_hmac_get_string()
}

Gets the HMAC as an hexadecimal string.

Once this function has been called the GLib.Hmac can no longer be updated with GLib.Hmac.prototype.update.

The hexadecimal characters will be lower case.

Since 2.30

Returns

the hexadecimal representation of the HMAC. The returned string is owned by the HMAC and should not be modified or freed.
