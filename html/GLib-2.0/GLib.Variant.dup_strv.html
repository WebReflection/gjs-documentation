GLib.Variant.prototype.dup_strv

function dup_strv(): [return_value: Array(String), length: Number(gsize)] {
    // Gjs wrapper for g_variant_dup_strv()
}

Gets the contents of an array of strings GLib.Variant. This call makes a deep copy; the return result should be released with GLib.strfreev.

If length is non-null then the number of elements in the result is stored there. In any case, the resulting array will be null-terminated.

For an empty array, length will be set to 0 and a pointer to a null pointer will be returned.

Since 2.24

return_value

an array of strings

length

the length of the result, or null
