GLib.OptionError

Error codes returned by option parsing.

OptionError.UNKNOWN_OPTION

An option was not known to the parser. This error will only be reported, if the parser hasn't been instructed to ignore unknown options, see GLib.OptionContext.prototype.set_ignore_unknown_options.

OptionError.BAD_VALUE

A value couldn't be parsed.

OptionError.FAILED

A GLib.OptionArgFunc callback failed.
