GLib.MainContext.default

function default(): GLib.MainContext {
    // Gjs wrapper for g_main_context_default()
}

Returns the global default main context. This is the main context used for main loop functions when a main loop is not explicitly specified, and corresponds to the "main" main loop. See also GLib.main_context_get_thread_default.

the global default main context.
