GLib.MainContext.prototype.release

function release(): void {
    // Gjs wrapper for g_main_context_release()
}

Releases ownership of a context previously acquired by this thread with GLib.MainContext.prototype.acquire. If the context was acquired multiple times, the ownership will be released only when GLib.MainContext.prototype.release is called as many times as it was acquired.
