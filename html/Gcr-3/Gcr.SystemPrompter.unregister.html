Gcr.SystemPrompter.prototype.unregister

function unregister(wait: Boolean): void {
    // Gjs wrapper for gcr_system_prompter_unregister()
}

Unregister this system prompter on the DBus @connection.

The prompter must have previously been registered with Gcr.SystemPrompter.prototype.register.

If wait is set then this function will wait until all prompts have been closed or cancelled. This is usually only used by tests.

wait

whether to wait for closing prompts
