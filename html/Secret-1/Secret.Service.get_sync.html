Secret.Service.get_sync

function get_sync(flags: Secret.ServiceFlags, cancellable: Gio.Cancellable): Secret.Service {
    // Gjs wrapper for secret_service_get_sync()
}

Get a Secret.Service proxy for the Secret Service. If such a proxy object already exists, then the same proxy is returned.

If flags contains any flags of which parts of the secret service to ensure are initialized, then those will be initialized before returning.

This method may block indefinitely and should not be used in user interface threads.

flags

flags for which service functionality to ensure is initialized

cancellable

optional cancellation object

Returns

a new reference to a Secret.Service proxy, which should be released with g_object_unref().
