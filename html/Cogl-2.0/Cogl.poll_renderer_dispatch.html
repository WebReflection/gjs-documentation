Cogl.poll_renderer_dispatch

function poll_renderer_dispatch(renderer: Cogl.Renderer, poll_fds: Cogl.PollFD, n_poll_fds: Number(gint)): void {
    // Gjs wrapper for cogl_poll_renderer_dispatch()
}

This should be called whenever an application is woken up from going idle in its main loop. The poll_fds array should contain a list of file descriptors matched with the events that occurred in revents. The events field is ignored. It is safe to pass in extra file descriptors that Cogl didn't request when calling Cogl.poll_renderer_get_info or a shorter array missing some file descriptors that Cogl requested.

<note>If your application didn't originally create a Cogl.Renderer manually then you can easily get a Cogl.Renderer pointer by calling cogl_get_renderer().</note>

Since 1.16

renderer

A Cogl.Renderer

poll_fds

An array of Cogl.PollFD<!-- -->s describing the events that have occurred since the application went idle.

n_poll_fds

The length of the poll_fds array.
