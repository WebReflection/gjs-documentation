Cogl.Framebuffer.prototype.clear

function clear(buffers: Number(gulong), color: Cogl.Color): void {
    // Gjs wrapper for cogl_framebuffer_clear()
}

Clears all the auxiliary buffers identified in the buffers mask, and if that includes the color buffer then the specified color is used.

Since 1.8

buffers

A mask of Cogl.BufferBit<!-- -->'s identifying which auxiliary buffers to clear

color

The color to clear the color buffer too if specified in buffers.
