Gcr.Prompt

const Gcr = imports.gi.Gcr;

let prompt = new Gcr.Prompt({
    caller_window: value,
    cancel_label: value,
    choice_label: value,
    continue_label: value,
    description: value,
    message: value,
    title: value,
    warning: value,
});

Represents a Gcr.Prompt displayed to the user.

Prerequisites
