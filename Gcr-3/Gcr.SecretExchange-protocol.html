Gcr.SecretExchange.protocol

SecretExchange.protocol: String (Read / Write / Construct Only)

The protocol being used for the exchange.

Will be null if no protocol was specified when creating this object, and either Gcr.SecretExchange.prototype.begin or Gcr.SecretExchange.prototype.receive have not been called successfully.
