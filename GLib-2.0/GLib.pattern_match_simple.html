Matches a string against a pattern given as a string. If this function is to be called in a loop, it's more efficient to compile the pattern once with GLib.PatternSpec.new and call GLib.pattern_match_string repeatedly.

pattern

the UTF-8 encoded pattern

string

the UTF-8 encoded string to match

Returns

true if string matches @pspec
