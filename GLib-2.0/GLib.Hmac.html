GLib.Hmac

const GLib = imports.gi.GLib;

let hmac = new GLib.Hmac();

An opaque structure representing a HMAC operation. To create a new GHmac, use GLib.Hmac.new. To free a GHmac, use GLib.Hmac.prototype.unref.
