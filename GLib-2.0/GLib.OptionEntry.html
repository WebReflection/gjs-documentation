GLib.OptionEntry

const GLib = imports.gi.GLib;

let optionEntry = new GLib.OptionEntry({
    short_name: value
    flags: value
    arg: value
});

A <structname>GOptionEntry</structname> defines a single option. To have an effect, they must be added to a GLib.OptionGroup with GLib.OptionContext.prototype.add_main_entries or GLib.OptionGroup.prototype.add_entries.
