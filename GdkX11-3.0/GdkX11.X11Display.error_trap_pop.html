GdkX11.X11Display.prototype.error_trap_pop

function error_trap_pop(): Number(gint) {
    // Gjs wrapper for gdk_x11_display_error_trap_pop()
}

Pops the error trap pushed by GdkX11.X11Display.prototype.error_trap_push. Will XSync() if necessary and will always block until the error is known to have occurred or not occurred, so the error code can be returned.

If you don’t need to use the return value, GdkX11.X11Display.prototype.error_trap_pop_ignored would be more efficient.

See gdk_error_trap_pop() for the all-displays-at-once equivalent.

Since 3.0

Returns

X error code or 0 on success
