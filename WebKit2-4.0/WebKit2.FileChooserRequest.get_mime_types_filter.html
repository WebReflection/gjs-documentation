WebKit2.FileChooserRequest.prototype.get_mime_types_filter

function get_mime_types_filter(): Gtk.FileFilter {
    // Gjs wrapper for webkit_file_chooser_request_get_mime_types_filter()
}

Get the filter currently associated with the request, ready to be used by Gtk.FileChooser. This function should normally be called before presenting the file chooser dialog to the user, to decide whether to apply a filter so the user would not be allowed to select files with other MIME types.

See WebKit2.FileChooserRequest.prototype.get_mime_types if you are interested in getting the list of accepted MIME types.

Returns

a Gtk.FileFilter if a list of accepted MIME types is defined or null otherwise. The returned object is owned by WebKitGTK+ should not be modified or freed.
