WebKit2.JavascriptResult.prototype.unref

function unref(): void {
    // Gjs wrapper for webkit_javascript_result_unref()
}

Atomically decrements the reference count of js_result by one. If the reference count drops to 0, all memory allocated by the WebKit2.JavascriptResult is released. This function is MT-safe and may be called from any thread.
