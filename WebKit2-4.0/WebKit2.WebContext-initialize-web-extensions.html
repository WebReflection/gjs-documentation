WebKit2.WebContext::initialize-web-extensions

connect('initialize-web-extensions', function (webContext, ): void);

This signal is emitted when a new web process is about to be launched. It signals the most appropriate moment to use WebKit2.WebContext.prototype.set_web_extensions_initialization_user_data and WebKit2.WebContext.prototype.set_web_extensions_directory.

Since 2.4

webContext

instance of WebKit2.WebContext that is emitting the signal
