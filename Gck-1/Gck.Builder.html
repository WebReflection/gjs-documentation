Gck.Builder

const Gck = imports.gi.Gck;

let builder = new Gck.Builder(flags: Gck.BuilderFlags);

A builder for a set of attributes. Add attributes to a builder, and then use Gck.Builder.prototype.end to get the completed Gck.Attributes.

The fields of Gck.Builder are private and not to be accessed directly.
