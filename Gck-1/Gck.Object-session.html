Gck.Object.session

Object.session: Gck.Session (Read / Write / Construct Only)

The PKCS11 session to make calls on when this object needs to perform operations on itself.

If this is NULL then a new session is opened for each operation, such as Gck.Object.get, Gck.Object.prototype.set or Gck.Object.prototype.destroy.
