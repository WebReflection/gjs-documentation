Gck.MechanismInfo

    min_key_size: value
    max_key_size: value
    flags: value

Represents information about a PKCS11 mechanism.

This is analogous to a CK_MECHANISM_INFO structure.

When you're done with this structure it should be released with Gck.MechanismInfo.prototype.free.
