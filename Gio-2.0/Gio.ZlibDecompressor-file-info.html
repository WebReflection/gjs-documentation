Gio.ZlibDecompressor.file_info

ZlibDecompressor.file_info: Gio.FileInfo (Read)

A Gio.FileInfo containing the information found in the GZIP header of the data stream processed, or null if the header was not yet fully processed, is not present at all, or the compressor's Gio.ZlibDecompressor.format property is not Gio.ZlibCompressorFormat.gzip.

Since 2.26
