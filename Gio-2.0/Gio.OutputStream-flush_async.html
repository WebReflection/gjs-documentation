Gio.OutputStream::flush_async

function vfunc_flush_async(io_priority: Number(gint), cancellable: Gio.Cancellable, callback: Gio.AsyncReadyCallback): void {
}

Forces an asynchronous write of all user-space buffered data for the given stream. For behaviour details see Gio.OutputStream.prototype.flush.

When the operation is finished callback will be called. You can then call Gio.OutputStream.prototype.flush_finish to get the result of the operation.

io_priority

the io priority of the request.

cancellable

optional Gio.Cancellable object, null to ignore.

callback

a Gio.AsyncReadyCallback to call when the request is satisfied
