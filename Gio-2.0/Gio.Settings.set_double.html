Gio.Settings.prototype.set_double

function set_double(key: String, value: Number(gdouble)): Boolean {
    // Gjs wrapper for g_settings_set_double()
}

Sets key in settings to value.

A convenience variant of Gio.Settings.set for doubles.

It is a programmer error to give a key that isn't specified as having a 'double' type in the schema for settings.

Since 2.26

key

the name of the key to set

value

the value to set it to

Returns

true if setting the key succeeded, false if the key was not writable
