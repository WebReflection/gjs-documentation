Gio.Cancellable.prototype.push_current

function push_current(): void {
    // Gjs wrapper for g_cancellable_push_current()
}

Pushes cancellable onto the cancellable stack. The current cancellable can then be received using Gio.Cancellable.get_current.

This is useful when implementing cancellable operations in code that does not allow you to pass down the cancellable object.

This is typically called automatically by e.g. Gio.File operations, so you rarely have to call this yourself.
