Gio.File::get_uri_scheme

function vfunc_get_uri_scheme(): String {
}

Gets the URI scheme for a Gio.File. RFC 3986 decodes the scheme as: |[ URI = scheme ":" hier-part [ "?" query ] [ "#" fragment ] ]| Common schemes include "file", "http", "ftp", etc.

This call does no blocking I/O.

Returns

a string containing the URI scheme for the given Gio.File. The returned string should be freed with GLib.free when no longer needed.
