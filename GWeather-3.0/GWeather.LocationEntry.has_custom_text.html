GWeather.LocationEntry.prototype.has_custom_text

function has_custom_text(): Boolean {
    // Gjs wrapper for gweather_location_entry_has_custom_text()
}

Checks whether or not entry's text has been modified by the user. Note that this does not mean that no location is associated with entry. GWeather.LocationEntry.prototype.get_location should be used for this.

Returns

true if entry's text was modified by the user, or false if it's set to the default text of a location.
