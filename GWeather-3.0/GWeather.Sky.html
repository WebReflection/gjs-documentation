GWeather.Sky

The sky and cloud visibility. In general it is discouraged to use this value directly to compute the forecast icon: applications should instead use GWeather.Info.prototype.get_icon_name or GWeather.Info.prototype.get_symbolic_icon_name.

Sky.INVALID

value not available

Sky.CLEAR

sky completely clear, no clouds visible

Sky.BROKEN

sky mostly clear, few clouds

Sky.SCATTERED

sky mostly clear, patches of clouds

Sky.FEW

few clouds, sky cloudy but patches of sky visible

Sky.OVERCAST

sky completely clouded, sun not visible

Sky.LAST

the maximum value for the enumeration
