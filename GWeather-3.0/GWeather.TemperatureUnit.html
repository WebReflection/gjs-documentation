GWeather.TemperatureUnit

The measure unit to use for temperature values, when retrieved by the GWeather.Info.prototype.get_value_temp family of functions.

TemperatureUnit.INVALID

invalid unit

TemperatureUnit.DEFAULT

the user preferred temperature unit

TemperatureUnit.KELVIN

Kelvin (absolute) temperature scale

TemperatureUnit.CENTIGRADE

Celsius temperature scale

TemperatureUnit.FAHRENHEIT

Fahrenheit temperature scale
