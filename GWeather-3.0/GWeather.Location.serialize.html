GWeather.Location.prototype.serialize

function serialize(): GLib.Variant {
    // Gjs wrapper for gweather_location_serialize()
}

Transforms a GWeather.Location into a GLib.Variant, in a way that calling GWeather.Location.prototype.deserialize will hold an equivalent GWeather.Location. The resulting variant can then be stored into GSettings or on disk. This call is only valid for cities, weather stations and detached locations. The format of the resulting GLib.Variant is private to libgweather, and it is subject to change. You should use the "v" format in GSettings, to ensure maximum compatibility with future versions of the library.

Returns

the serialization of @location.
