GObject.BindingFlags

Flags to be passed to GObject.Object.prototype.bind_property or GObject.Object.prototype.bind_property_full.

This enumeration can be extended at later date.

Since 2.26

BindingFlags.DEFAULT

The default binding; if the source property changes, the target property is updated with its value.

BindingFlags.BIDIRECTIONAL

Bidirectional binding; if either the property of the source or the property of the target changes, the other is updated.

BindingFlags.SYNC_CREATE

Synchronize the values of the source and target properties when creating the binding; the direction of the synchronization is always from the source to the target.

BindingFlags.INVERT_BOOLEAN

If the two properties being bound are booleans, setting one to true will result in the other being set to false and vice versa. This flag will only work for boolean properties, and cannot be used when passing custom transformation functions to GObject.Object.prototype.bind_property_full.
